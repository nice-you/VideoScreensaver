# 7981 屏保

> 一个能自动下载 B 站视频、轮播作为电脑动态屏保的小工具

## 功能

- 自动缓存 B 站热门视频（仅个人学习使用），离线也能循环播放
- 自动下载 picsum 高清壁纸作为背景图库
- 在无操作的五分钟后打开自动屏保窗口
- 进度条支持随意跳转，Esc 关闭窗口
- Ctrl+Shift+E 安全退出程序

## 快速开始

1. 下载 [最新 Release](https://github.com/nice-you/VideoScreensaver/releases) 里的 `7981Screensaver.exe`
2. 解压到任意文件夹，双击运行即可（会自动生成 `backgrounds` 和 `cache` 文件夹）
